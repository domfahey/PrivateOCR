# Local OCR Screenshot Project

This repository contains the source code for the **Local OCR Screenshot** Chrome extension.

## Directory Structure

-   **`ocr-local-screenshot-ext/`**: The main extension directory. Contains source code, tests, and configuration.
    -   `src/`: Extension source code (popup, background, content scripts).
    -   `vendor/`: Bundled dependencies ([Tesseract.js](https://github.com/naptha/tesseract.js)).
    -   `test/`: Unit tests.
-   **`INSTALL.md`**: Detailed installation guide for users.
-   **`PUBLISHING.md`**: Guide for publishing the extension to the Chrome Web Store.

## Getting Started

Please refer to [ocr-local-screenshot-ext/README.md](ocr-local-screenshot-ext/README.md) for details on the extension's features and usage.

For developer contributions, see [ocr-local-screenshot-ext/CONTRIBUTING.md](ocr-local-screenshot-ext/CONTRIBUTING.md).
